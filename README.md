# PSUSphere

## Description
PSUSphere is a Django web application designed to manage student organizations, programs, and colleges. It allows users to register students, track organization memberships, and view associated programs and colleges.

## Features
- Add and manage Colleges, Programs, and Organizations
- Register Students and assign them to Programs
- Track student memberships in Organizations
- Generate fake data for testing using Faker
- Admin interface for easy management
